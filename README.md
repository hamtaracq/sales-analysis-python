# Sales Analysis with Python

## Overview
This project analyzes sales data using Python and Pandas.

## Tools Used
- Python
- Pandas
- Matplotlib

## Tasks Performed
- Grouped sales by country
- Calculated average sales by product
- Found the top-selling country
- Created a bar chart visualization

## Files
- sales.csv
- sales_analysis.ipynb

## Key Results
- Germany had the highest total sales
- Sales trends were visualized using a bar chart
